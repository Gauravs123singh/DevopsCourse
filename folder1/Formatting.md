# Formatting in GitHub repository

**This is a bold line**

*This is italic and bold*

_This is italic_

## This is heading 2

### This is heading 3

**This is for ordered list:**

1. Point 1
2. Point 2
3. Point 3
   1. Subpoint 1
   2. Subpoint 2

**This is for unordered list:**

* Point 1
* Point 2
* Point 3
  * Subpoint 1
  * Subpoint 2

> **NOTE** : It is a note
